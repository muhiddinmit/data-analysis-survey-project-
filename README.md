# 📊 Developer Survey Data Analysis Project

## 🚀 Overview
This project analyzes developer survey data to find insights about salary, experience, and technologies.

## 🎯 Objectives
- Clean and prepare data
- Analyze trends
- Build dashboard

## 🧹 Data Cleaning
- Handled missing values
- Removed unnecessary columns

## 📊 Analysis
- Salary by country
- Experience vs salary
- Popular technologies

## 📈 Dashboard
Built interactive dashboard using Power BI.

## 🛠️ Tools
- Python
- Pandas
- Power BI
